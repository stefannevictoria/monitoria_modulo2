## 🛍️ O Projeto de Exemplo: ShopEasy

Para que todos os exemplos se conectem, usamos um único domínio ao longo de todos os documentos.

**O que o ShopEasy faz:**
- Usuários se cadastram e fazem login
- Produtos são organizados em categorias
- Usuários adicionam produtos ao carrinho e fazem pedidos
- Cada pedido tem itens, e cada item tem quantidade e preço

**Regras de Negócio principais:**
- Um pedido deve ter pelo menos 1 item
- O estoque de um produto não pode ficar negativo
- Um usuário pode ter vários endereços, mas só um padrão
- Pedidos cancelados não podem ser reabertos
